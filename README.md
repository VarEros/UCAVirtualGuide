## Uso libre
Este proyecto está abierto a ser modificado con motivos de aprendizaje, se deben dejar los créditos respectivos a los autores intelectuales de este proyecto.

## Detalles
Este proyecto será presentado durante feria de carreras de la [Universidad Centroamericana de Nicaragua](https://www.uca.edu.ni/), este ha sido desarrollado en un período de nueve días bajo el motor Unity en C# utilizando como aditivo a la realidad virtual [Google Cardboard XR Plugin](https://github.com/googlevr/cardboard) y demás herramientas multimedia. Por otra parte, este proyecto tiene como objetivo una experiencia semi-inmersiva durante el paso de la vista panorámica en algunas edificaciones del alma mater.

Este proyecto pretende mostrar el músculo de Ingeniería en Sistemas de Información, haablando de las metodologías, pensamiento y lógica que hemos desarrollado en dos años de carrera.
